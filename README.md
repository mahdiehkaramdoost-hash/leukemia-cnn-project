# leukemia-cnn-project
# 🧠 تشخیص لوسمی با استفاده از شبکه عصبی کانولوشنی (CNN)

## 📌 معرفی پروژه
این پروژه یک مدل یادگیری عمیق مبتنی بر شبکه عصبی کانولوشنی (CNN) برای تشخیص بیماری لوسمی از روی تصاویر میکروسکوپی خون می‌باشد.

هدف این پروژه، طبقه‌بندی تصاویر به دو دسته زیر است:
- سلول‌های سالم (Healthy)
- سلول‌های مبتلا به لوسمی (Leukemia)

این پروژه به عنوان پایان‌نامه کارشناسی با استفاده از Google Colab پیاده‌سازی شده است.

---

## 📊 مجموعه داده (Dataset)
در این پروژه از مجموعه داده **ALL-IDB (Acute Lymphoblastic Leukemia Image Database)** استفاده شده است.

این دیتاست شامل سه بخش زیر است:
- L1
- L2
- L3

هر بخش شامل تصاویر میکروسکوپی سلول‌های خونی است.

---

## 🧠 معماری مدل
مدل استفاده شده یک شبکه عصبی کانولوشنی (CNN) است که شامل بخش‌های زیر می‌باشد:

- لایه‌های کانولوشن (Conv2D) برای استخراج ویژگی‌ها
- لایه‌های MaxPooling برای کاهش ابعاد
- لایه Flatten
- لایه‌های Fully Connected (Dense)
- لایه خروجی:
  - `Dense(1, activation='sigmoid')` برای طبقه‌بندی دودویی

---

## 📈 نتایج آموزش مدل

- دقت اعتبارسنجی (Validation Accuracy): حدود 100٪

📌 توجه: به دلیل کوچک بودن دیتاست، احتمال بیش‌برازش (Overfitting) وجود دارد.

---

## 📉 معیارهای ارزیابی
برای ارزیابی مدل از معیارهای زیر استفاده شده است:

- Precision
- Recall
- F1-score
- Confusion Matrix
